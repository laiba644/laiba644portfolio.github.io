## 🚀 How to Deploy FREE (Step-by-Step)

### Option 1: GitHub Pages (RECOMMENDED — 100% Free)
**Your URL will be:** `https://laiba644.github.io/portfolio`

1. Go to [github.com](https://github.com) — log in with your account
2. Click **"New Repository"** → name it exactly: `portfolio`
3. Set it to **Public** → Click **"Create repository"**
4. Upload all 3 files (`index.html`, `style.css`, `script.js`) by clicking **"uploading an existing file"**
5. Commit the files
6. Go to **Settings → Pages → Source → Deploy from branch → main**
7. Wait 2-3 minutes → Your site is live! ✅
8. Your link: `https://laiba644.github.io/portfolio`

---

### Option 2: Netlify (Drag & Drop — Even Easier!)
**Your URL will be:** `https://laiba-kashif.netlify.app` (you choose the name)

1. Go to [netlify.com](https://netlify.com) → Sign up free
2. Click **"Add new site" → "Deploy manually"**
3. Drag the entire portfolio folder into the box
4. Site goes live instantly! ✅
5. Click **"Domain settings"** to give it a custom name like `laibakashif.netlify.app`

---

### Option 3: Vercel
1. Go to [vercel.com](https://vercel.com) → Sign up with GitHub
2. Click **"Add New → Project"**
3. Import your GitHub repo OR drag and drop the folder
4. Deploy — done! ✅

## ✏️ How to Edit Content

### Change your name or text:
Open `index.html` in VS Code and edit the text directly.

### Change colors:
Open `style.css` → find `:root` at the top → change `--accent` to any color.
Example: `--accent: #FF6B6B;` for red, `--accent: #7DF9C2;` for current green.

### Add a new project:
Copy a `.project-card` block in `index.html` and paste it, changing the content.

### Update GitHub links:
Search for `https://github.com/laiba644` in `index.html` and replace with your specific repo URLs.
